# Sounds

Optional audio files for the BINGO / IKKE BINGO buttons. The app works
without them (it falls back to synthesized sounds), and picks them up
automatically once present.

| File | Plays on | Notes |
|------|----------|-------|
| `bingo-song.mp3` | **BINGO** (win) | Only the **first 20 seconds** are played. Falls back to a synthesized fanfare if missing. |
| `not-bingo.mp3`  | **IKKE BINGO** (lose) | Falls back to a synthesized "sad trombone" if missing. |

Format: `.mp3`. Keep the exact filenames above.

## ⚠️ Licensing

This app is deployed publicly. Only add audio you have the right to
distribute. A commercial track such as "Bingo Song – Ashley Wheelhouse"
is copyrighted; publicly streaming even a short excerpt may require a
licence. That's your responsibility — the code just plays whatever file
you place here.
